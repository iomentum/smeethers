# Install

Clone the project and install back and front dependencies

_in your workspace:_

```
git clone https://github.com/iomentum/smeethers.git
cd ./smeethers/smeethersApp
npm i
cd ../backend
npm i
cd ../
```

# Run the project

## Front

_from the root folder:_

```
cd ./smeethersApp
```

### Android

```
npm run android
```

### iOs

```
npm run ios
```

## Smeethers backend

_from the root folder:_

```
npm start
```
